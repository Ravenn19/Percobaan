## Tentang Projek

**TechnoPark STP ITPLN** adalah sistem manajemen terintegrasi untuk TechnoPark di Sekolah Tinggi Teknologi PLN. Projek ini dibangun dengan:

- Laravel 10.x
- Livewire 3.x
- Tailwind CSS
- MySQL Database

Fitur utama:
- Manajemen tenant TechnoPark
- Sistem peminjaman fasilitas
- Pelaporan kegiatan
- Integrasi dengan sistem akademik
- Dashboard administrasi
